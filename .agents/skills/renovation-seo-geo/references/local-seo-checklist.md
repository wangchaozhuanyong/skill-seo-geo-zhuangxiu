# Local SEO Checklist

Check:

- NAP consistency
- Google Business Profile data only if owner provided
- Baidu local/map data only if owner provided
- verified service areas only
- city pages have unique local context
- no city-swap doorway pages
- local CTA uses owner-approved phone/WhatsApp
- reviews only if real
- local photos only if real
- concept images clearly labeled
- citation opportunities are reviewed before submission

Do not create local pages for unsupported areas.
