# Content Style Guide

## Language

Default owner-facing language: clear Simplified Chinese.
If the target page is English, write the analysis, approval notes, QA, and daily report in Simplified Chinese, and place publishable English page copy in a clearly labeled English section.
If the target page is Chinese, write both the review document and publishable page copy in clear Simplified Chinese.
If the site targets Malaysia/Singapore, avoid overly Mainland-only phrasing unless the brand uses it.

## Style

- Clear
- Practical
- Specific
- Trustworthy
- Local where relevant
- Helpful to homeowners and business owners

## Avoid

- Keyword stuffing
- Empty marketing words
- Fake urgency
- Fake discounts
- Unsupported superlatives
- Long generic introductions
- Repeating the same city/service phrase unnaturally

## Preferred structure

- Direct answer or summary near top
- Clear H2 sections
- Short paragraphs
- Practical examples
- Clearly labeled design concepts or effect-rendering examples when real cases are unavailable
- FAQ
- CTA
- Internal links

## Renovation content should explain

- Design concept
- Effect-rendering or 3D visualization direction when useful
- Layout planning
- Budget factors
- Timeline factors
- Material choices
- Process
- Common mistakes
- Risks and limitations
- What the customer should prepare
- When to contact a professional
