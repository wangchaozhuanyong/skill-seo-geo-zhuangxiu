# Publishing Checklist

Before publishing any SEO/GEO content, check:

## Publishing gate

- Specific draft or optimization plan approved by owner
- Owner explicitly asked Codex to execute this exact draft or plan
- Target page or CMS entry confirmed
- Language scope confirmed; default is both `/zh` and `/en` page versions for page-level SEO/GEO execution
- Publishing method confirmed: CMS/admin or source edit
- Not part of an unattended scheduled automation
- First 2-4 week draft-only observation period considered before publishing
- No admin credentials stored in prompts, drafts, reports, or data files

## Business accuracy

- Company name correct
- Service actually offered
- Location actually served
- Contact details correct
- No fake price
- No fake project
- No fake testimonial
- No fake certification
- No unsupported guarantee
- Design concept, rendering, or planning examples are clearly labeled if used
- Concept/rendering content is not described as a completed real project, customer review, before/after proof, fixed price, fixed timeline, or warranty promise
- Warranty scope and duration confirmed if mentioned
- Years in business confirmed if mentioned
- SSM/license/certification details confirmed if mentioned

## SEO

- One clear primary keyword
- Natural secondary keywords
- Useful title
- Useful meta description
- One clear H1
- Logical H2/H3
- Internal links
- Image alt text
- Schema only if accurate
- No keyword stuffing
- Not duplicate/thin content
- Not a city-name swap page

## GEO / AI search

- Clear answer near top
- Practical examples
- Specific decision factors
- FAQ
- Real experience or proof where possible
- Easy to summarize
- No artificial "AI bait"
- No page created only to manipulate AI search mentions
- Real services, areas, cases, materials, process, and FAQ used where available
- Concept/rendering material clearly labeled when real case material is unavailable

## Conversion

- CTA visible
- Contact path clear
- Suitable offer
- Trust signals included
- Next step obvious
- CTA wording and phone/WhatsApp/contact URL confirmed

## Technical

- Correct URL
- Both Chinese and English URLs updated when bilingual execution is approved
- Page indexable if intended
- No accidental noindex
- No broken images
- No broken internal links
- Mobile readable
- Fast enough
- Included in sitemap if published
- Edge SEO metadata, schema, sitemap, and SEO manifest regenerated/deployed when the website uses a static or edge SEO layer
