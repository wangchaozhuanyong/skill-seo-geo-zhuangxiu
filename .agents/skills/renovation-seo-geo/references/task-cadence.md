# SEO/GEO Task Cadence

## Daily

Choose one high-value task:

- service page optimization
- case study draft
- local page draft
- old article refresh
- metadata improvement
- FAQ improvement
- internal linking
- image alt text
- schema suggestion
- technical SEO scan
- daily brief

## Weekly

- Search Console review when exports are available
- keyword opportunity report
- high-impression low-CTR report
- ranking position 4-20 opportunity report
- content gap review
- internal link review
- weekly SEO/GEO report

## Monthly

- content pruning recommendations
- service page audit
- local SEO audit
- case study coverage audit
- technical SEO audit
- 30-day roadmap

## Suggested weekly rhythm

- Monday: core service page optimization
- Tuesday: real case study draft or improvement
- Wednesday: old content refresh
- Thursday: local SEO page or service-area improvement
- Friday: internal links, metadata, schema, and image alt text
- Saturday: Search Console or analytics review if exports are available
- Sunday: next-week plan and owner input list

This rhythm is a guide, not a rule. The daily task should still be chosen by value, data, and available business facts.
