# GEO / AI Search Checklist

GEO work should improve clarity for real users, search engines, and AI search systems.

Check:

- brand/entity consistency
- company name consistency
- NAP consistency
- service taxonomy
- verified area coverage
- direct answer blocks
- concise summary blocks
- evidence blocks
- FAQ
- schema
- crawlability
- indexability
- page clarity
- unique non-commodity content
- concept/rendering labels

Do not create AI bait, mass query-variation pages, fake citations, fake cases, fake reviews, awards, or media claims.
