# Multilingual SEO Checklist

Default language scope is bilingual.

Check:

- `/zh` and `/en` URL pair
- hreflang self reference
- hreflang alternate reference
- canonical does not cross language
- sitemap includes both language URLs
- Chinese page has Chinese content
- English page has English content
- title/meta match the same intent in both languages
- service slug pair is consistent
- historical aliases have redirect/canonical strategy

Only create a single-language plan when the owner explicitly asks for one.
