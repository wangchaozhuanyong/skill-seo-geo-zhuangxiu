# Technical SEO Checklist

Check:

- HTTP status
- indexability
- robots allowed
- canonical self-reference
- no cross-language canonical
- sitemap inclusion
- hreflang pair
- title present
- meta description present
- H1 present
- schema types
- internal inlinks and outlinks
- image count and missing alt
- page word count
- priority issue

Blocking issues before publish:

- non-200 target
- noindex target
- robots blocked target
- wrong canonical
- missing title/meta/H1
- missing paired language page when bilingual update is required
- schema validation errors
